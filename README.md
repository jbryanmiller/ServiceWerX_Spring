# ServiceWerX_Spring

Spring Boot Example + REST + JPA
---------------------------------------------

This project is an example Spring Boot + REST + JPA Application

How to run application
---------------------------------------------
```bash
## Build application from base directory
mvn clean install

## Run Spring boot application with embedded H2 database
mvn spring-boot:run
```
